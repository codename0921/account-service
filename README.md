# account-service 🪙
스프링 프레임워크를 활용한 은행 계좌 이용 서비스

## 프로젝트 소개
스프링을 사용해 로그인/로그아웃 기능을 구현하여 계좌를 접근하고, 이외 계좌 생성, 삭제 등 계좌 관리 기능을 제공하며, 추가적으로 입출금 및 송금 기능을 구현한 간단한 은행 계좌 서비스입니다.

## 개발 기간
2023.06.12 ~ 2023.07.17

## 개발 환경
- IntelliJ IDEA Community Edition
- GIT
- GITHUB

- JAVA
- SPRING FRAMEWORK

## 주요 기능
**1. 계좌 검색 기능**
- [ ]  보유 계좌 검색

**2. 계좌 관리**
- [ ] 계좌 생성
- [ ] 계좌 삭제
- [ ] 인출
- [ ] 입금

**3. 송금 및 송금 이력 조회**
- [ ] 계좌별 기간 내 송금 내역 조회

**4. 계좌 접근 허가 기능**
- [ ] 유저 로그인/로그아웃
